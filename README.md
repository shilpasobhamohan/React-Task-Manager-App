# React Task Manager Application

A simple full-stack task management application built using React and .NET Web API.
“Recreated the same application in Angular and React”

## Features
- Add and delete tasks
- Real-time UI updates
- REST API integration

## Tech Stack
- Frontend: React (useState, useEffect)
- Backend: .NET Web API
- Database: SQLite
